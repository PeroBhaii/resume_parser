API Documentation

Learn how to request data from the API.

Request Data

To request data from the API, you can use the following code example in Python using the requests library:

import requests

url = 'http://127.0.0.1:5000/parse'
files = {'resume': ('ATS_classic_HR_resume.pdf', open('static/ATS_classic_HR_resume.pdf', 'rb'))}
response = requests.post(url, files=files)

if response.status_code == 200:
    data = response.json()
    print(data)
else:
    print("Error:", response.status_code)

Explanation

This code sends a POST request to the API endpoint (/parse) with a resume file attached as a multipart/form-data request. If the response status code is 200, the API has successfully processed the request, and you can access the parsed data in the data variable.

API Response

The API will return a JSON response containing the parsed data. You can access this data for further processing as needed.

Additional Information

For more information on API usage and available endpoints, please refer to the API documentation or contact the API provider.
